# U&N Werkstatt-Quiz

Interaktives Offline-Quiz zum Lernfeld **Um- und Nachrüsten**.

## Start

Die Datei `index.html` in einem aktuellen Browser öffnen. Eine Installation und eine Internetverbindung sind nicht erforderlich.

## Umfang

- 29 Aufgaben
- 31:00 Minuten maximale Bearbeitungszeit
- 22.000 maximal erreichbare Punkte
- Zuordnungsaufgaben, Single-Choice und interaktive Diagrammauswahl
- Anteilige Punkte für jede richtige Einzelzuordnung
- Volle Punktzahl während der ersten 10 Sekunden, danach zeitabhängige Reduzierung
- Lokale Ergebnis- und Aufgabenübersicht mit Aufforderung zum Screenshot der Punktzahl

Alle Teilnehmerdaten und Ergebnisse bleiben ausschließlich im geöffneten Browserfenster und werden nicht übertragen oder gespeichert.
